# Auth Page 

This project was bootstrapped with Create React App. 

## Used Technologies

- Type Script
- React, useState, useEffect
- Redux
- Material UI
- Firebase (as backend service )

## Start service

First, install needful dependencies:

### `npm install`

In the project directory, you can run:

### `npm run start`
